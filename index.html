Intro

Installable via npm install --save datastore-pubsub, it can also be used directly in the browser.

Download

The source is available for download from GitHub. Alternatively, you can install using npm:

$ npm install --save datastore-pubsub

You can then require() datastore-pubsub as normal:

const datastorePubsub = require('datastore-pubsub')

In the Browser

Datastore-pubsub should work in any ES2015 environment out of the box.

Usage:

<script type="text/javascript" src="index.js"></script>

The portable versions of datastore-pubsub, including index.js and index.min.js, are included in the /dist folder. Datastore-pubsub can also be found on unpkg.com under

  • https://unpkg.com/datastore-pubsub/dist/index.min.js
  • https://unpkg.com/datastore-pubsub/dist/index.js
